# Kasparro AI Agentic Content Generation System

A multi-agent system for generating marketing content for skincare products using **LangChain** and **LangGraph**.

## Architecture
This project uses a graph-based agent workflow to orchestrate content generation.
See [Architecture Documentation](docs/architecture.md) for details.

## Addressing Reviewer Feedback
This v2 implementation explicitly solves the issues raised in the previous review:

| Critique | Solution in v2 |
| :--- | :--- |
| **"No framework-based agent boundaries"** | Replaced custom classes with **LangGraph Nodes**. Each agent is now a distinct node in a state graph, with clear input/output schemas defined by `AgentState`. |
| **"Pipeline behaves like a static script"** | Implemented a **StateGraph** workflow where execution flow is managed by the framework, not a linear script. |
| **"Content generation is hardcoded"** | Removed all hardcoded strings. Agents now use **Gemini-2.5-flash** via `PromptTemplate` to generate dynamic content. |
| **"No tool usage demonstrated"** | Utilized LangChain's `JsonOutputParser` and `Pydantic` schemas as tools to enforce structured output from the LLM. |

## Features
- **Framework-Native Orchestration**: Uses `LangGraph` for state management.
- **Model-Driven Content**: All text is generated by `gemini-2.5-flash`.
- **Structured Output**: Enforces valid JSON using Pydantic parsers.
- **Component Modularity**: Agents are isolated components with single responsibilities.

## Prerequisites
- Python 3.9+
- `GOOGLE_API_KEY` environment variable (for Gemini models)

## Setup
1. Clone the repository.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Setup your environment:
   Create a `.env` file in the root directory and add your API key:
   ```env
   GOOGLE_API_KEY=your_api_key_here
   ```

## Usage
Run the main script with an input file:
```bash
python main.py --input data/input_product.json --output output
```

## Agents
- `DataParserAgent`: Standardizes input.
- `QuestionGeneratorAgent`: Creates FAQs.
- `CompetitorGenerationAgent`: Simulates market research.
- `ContentDraftingAgent`: Writes copy.
- `PageAssemblerAgent`: Builds final JSON.
- `ValidationAgent`: Reviews output.
